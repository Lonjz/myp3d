# Quick Run Scripts

These scripts let you start the project quickly from the repository root.

## Windows

Start backend + frontend in separate PowerShell windows:

```powershell
.\run\dev-win.ps1
```

## macOS (or Linux)

Make scripts executable once:

```bash
chmod +x ./run/dev-mac.sh
```

Start backend + frontend:

```bash
./run/dev-mac.sh
```
